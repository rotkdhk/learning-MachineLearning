Few examples of Supervised Learning:

| Input | Output | Application |
|-------|--------|-------------|
| Email text & features | Spam / Not Spam | Email filtering |
| House size, location, rooms | Price (number) | Price prediction |
| Medical scan / test results | Disease / No disease | Medical diagnosis |
| Past transactions | Fraud / Not Fraud | Fraud detection |

Supervised Learning has 2 types:
- Regression
- Classification

## Regression

Predicting a number from infinitely many possible outputs. Let's say we have data like Temperature vs Number of customers, now we can predict, if we give `x` temperature, approximately how many customers can visit tomorrow.

## Classification

Predicting a category from a small number of possible outcomes. Like whether a picture is a Dog or Cat, or SPAM or NOT SPAM.
